#Football Pickem Pools with Rails & AngularJS

Steps to set up:

- Clone app
- ```Bundle install```
- ```rake bower:install```
- ```rake db:setup``` (right now this only creates a single league for testing and viewing purposes)
- ```rails s```
- Go to localhost:3000 and see angular in action
